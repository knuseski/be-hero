Prvo mora da instaliras node verzija 10

Posle treba da isntaliras mySQL server
Tamu ce kreiras tabeli i ce popolnis igri, imas primer bo SQL.sql fileto

vo .env-example smeni gi parametrite od tvojot sql i ce go renajmnes filot od .env-example vo .env

Kje otvoris terminal i ce navigiras do /server. Tuka ce pises 'npm install' da ti gi spusti site packages

Otkako ce se spustat site packages ce pises 'npm start' za da go ranuvas backendot.
Ako e se kako sto treba kje dobies message:

------------------------------------------------
[nodemon] 2.0.4
[nodemon] to restart at any time, enter `rs`
[nodemon] watching path(s): *.*
[nodemon] watching extensions: js,mjs,json
[nodemon] starting `node index.js`

Server is running on port 3000
Connected to mySQL!
----------------------------------------------

Frontendot e vo client folderot. Otvori index.html
